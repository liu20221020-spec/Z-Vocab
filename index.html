云端同步设置

用 Supabase 免费数据库同步 iPad、手机、电脑数据。

设置步骤：
1. 打开 supabase.com，免费注册并新建项目
2. 进入项目 → SQL Editor → 粘贴下方代码运行：

create table if not exists vocab_cards ( sync_key text not null, data jsonb not null, updated_at timestamptz default now(), primary key (sync_key) ); alter table vocab_cards disable row level security;
3. 进入 Project Settings → API，复制 Project URL 和 anon public key 填入下方
